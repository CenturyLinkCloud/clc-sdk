# baremetal-info

enumerates available baremetal configuration by datacenter. 

provides both SKU codes as well as available OS types.



    Usage of baremetal-info:
      -dc string
            datacenter
      -password string
            clc password
      -username string
            clc username

## invocation

`./baremetal-info -username XYZ -password XYZ -dc DC`
